metadata_cache
최대 100개 항목 · TTL 24시간
센서 목록, 데이터 범위 정보
replay_cache
최대 2,000개 항목 · 가변 TTL
오늘 데이터: 5분 | 과거 데이터: 7일
daterange_cache
최대 500개 항목 · TTL 24시간

기술 스택 & 라이브러리

사용 기술과 선택 이유

⚡ FastAPI
Python 비동기 웹 프레임워크. 자동 API 문서 생성, 타입 힌트 기반 검증, 고성능 비동기 처리
Backend
📊 D3.js v7
SVG 기반 데이터 시각화. 서울시 GeoJSON 지도 렌더링, 데이터 바인딩으로 동적 업데이트
Frontend
🗃 MySQL
안정적인 RDBMS. 인덱스 최적화로 대용량 시계열 데이터 빠른 조회, 트랜잭션 안정성
Database
🐍 PyMySQL
순수 Python MySQL 드라이버. C 확장 없이 설치 간편, 호환성 우수
Driver
🏊 DBUtils PooledDB
데이터베이스 연결 풀 관리. 동시 요청 처리, 연결 재사용으로 성능 향상
Pool
🌐 httpx
비동기 HTTP 클라이언트. S-DoT Open API 프록시 호출에 사용, CORS 우회
HTTP
🚀 Uvicorn
ASGI 서버. 비동기 워커로 높은 동시성 처리, FastAPI의 공식 권장 서버
ASGI
📝 Vanilla JS
프레임워크 오버헤드 제거. 빌드 불필요, 직접 DOM 제어로 경량하고 빠른 로딩
Frontend
